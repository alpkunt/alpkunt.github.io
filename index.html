About the series

One idea, one page, one conversation. Every project here is a single self‑contained HTML file — no servers, no build step, no tracking. It runs entirely in your browser and works offline once loaded.

The atlases share a method: things are scored on a set of comparable traits, standardized, and compared by distance, so "close" means "genuinely alike." The scores are an informed heuristic, not a measurement — read each atlas as a conversation starter, never a verdict.
